# レベルアイコン（8枚）

レベルに応じて表示されるアイコン画像をここに置きます。
ファイルが無い間は絵文字（emoji）が代わりに表示されます。
PNG を置くと自動で切り替わります。

| ファイル名 | 対象レベル | 目安の累計時間 | 仮の絵文字 |
|-----------|-----------|--------------|-----------|
| `tier1.png` | Lv 1–9     | 0〜2時間      | 🌱 |
| `tier2.png` | Lv 10–24   | 2〜12時間     | 📗 |
| `tier3.png` | Lv 25–49   | 12〜48時間    | ✏️ |
| `tier4.png` | Lv 50–99   | 48〜197時間   | 📘 |
| `tier5.png` | Lv 100–199 | 197〜795時間  | 🎓 |
| `tier6.png` | Lv 200–349 | 795〜2440時間 | 🔮 |
| `tier7.png` | Lv 350–499 | 2440〜5000時間| 👑 |
| `tier8.png` | Lv 500+    | 5000時間〜    | 🏆 |

## 仕様
- 正方形（推奨 256×256）の透過 PNG
- レベルが進化していく雰囲気で（例: 🌱→🌿→🌳…）
- レベルに上限は無いので `tier8.png` は Lv500 以降ずっと使われます
