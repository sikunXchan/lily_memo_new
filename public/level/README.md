# レベルアイコン（tier1〜tier15）

`public/level_icon/` に置かれた元画像（白背景のシート）から
`scripts/slice_level_icons.py` で自動スライスした個別アイコンです。
レベルに応じて表示されます。エフェクトは Tier 9 から始まり、上に行くほど
ド派手になり、Tier 15 で全虹（最大演出）になります。

レベルアップは**定速**で、**1レベル＝2時間**固定（`src/lib/level.ts` の
`HOURS_PER_LEVEL`）。下の累計時間はこの前提での目安です。

| ファイル | 対象レベル | 目安の累計時間 | 見た目 / エフェクト |
|---------|-----------|--------------|-----------|
| `tier1.png`  | Lv 1–3     | 0〜6時間       | 🥉 ブロンズ |
| `tier2.png`  | Lv 4–7     | 6〜14時間      | 🥉 ブロンズ |
| `tier3.png`  | Lv 8–12    | 14〜24時間     | 🥈 シルバー |
| `tier4.png`  | Lv 13–18   | 24〜36時間     | 🥈 シルバー |
| `tier5.png`  | Lv 19–26   | 36〜52時間     | 🥈 シルバー |
| `tier6.png`  | Lv 27–36   | 52〜72時間     | 🥇 ゴールド |
| `tier7.png`  | Lv 37–48   | 72〜96時間     | 🥇 ゴールド |
| `tier8.png`  | Lv 49–62   | 96〜124時間    | 🥇 ゴールド |
| `tier9.png`  | Lv 63–80   | 124〜160時間   | 💎 クリスタル ✨ 氷の発光 |
| `tier10.png` | Lv 81–102  | 160〜204時間   | 💠 クリスタル ✨ 強発光＋浮遊 |
| `tier11.png` | Lv 103–129 | 204〜258時間   | 💚 エメラルド 🌿 翠オーラ＋浮遊 |
| `tier12.png` | Lv 130–162 | 258〜324時間   | 👑 エメラルド 🌿 翠放射＋煌めき |
| `tier13.png` | Lv 163–202 | 324〜404時間   | 🪽 白銀 🌟 聖なるリング＋浮遊 |
| `tier14.png` | Lv 203–250 | 404〜500時間   | 😇 白銀 🌟 回転ハロー＋煌めき |
| `tier15.png` | Lv 251+    | 500時間〜      | 🌈 虹 ✨✨ 全虹（回転＋脈動＋煌めき、最大演出） |

- レベルに上限は無いので `tier15.png` は Lv251 以降ずっと使われます（永遠の頂点）。
- 区切り（minLevel）・色・エフェクトは `src/lib/level.ts` の `LEVEL_TIERS` で調整できます。
- 元画像を差し替えたら `python3 scripts/slice_level_icons.py` で再スライスしてください。
