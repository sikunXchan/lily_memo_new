# レベルアイコン（tier1〜tier9）

`public/level_icon/` に置かれた元画像（シート）から自動でスライスした
個別アイコンです。レベルに応じて表示されます。

| ファイル | 対象レベル | 目安の累計時間 | エフェクト |
|---------|-----------|--------------|-----------|
| `tier1.png` | Lv 1–19    | 0〜7時間       | なし |
| `tier2.png` | Lv 20–59   | 7〜70時間      | なし |
| `tier3.png` | Lv 60–109  | 70〜239時間    | なし |
| `tier4.png` | Lv 110–174 | 239〜609時間   | なし |
| `tier5.png` | Lv 175–249 | 609〜1246時間  | なし |
| `tier6.png` | Lv 250–329 | 1246〜2175時間 | ✨ ゴールド発光 |
| `tier7.png` | Lv 330–419 | 2175〜3528時間 | ✨ 強発光（金＋赤） |
| `tier8.png` | Lv 420–499 | 3528〜5000時間 | 🌟 放射オーラ＋浮遊 |
| `tier9.png` | Lv 500+    | 5000時間〜     | 🌈 虹ハロー＋浮遊 |

- レベルに上限は無いので `tier9.png` は Lv500 以降ずっと使われます。
- 区切り（minLevel）やエフェクトは `src/lib/level.ts` の `LEVEL_TIERS` で調整できます。
- 元画像を差し替えたら `scripts/slice_level_icons.py` 相当で再スライスしてください。
