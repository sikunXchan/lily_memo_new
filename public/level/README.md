# レベルアイコン（tier1〜tier15）

`public/level_icon/` に置かれた元画像（白背景のシート）から
`scripts/slice_level_icons.py` で自動スライスした個別アイコンです。
レベルに応じて表示されます。エフェクトは Tier 9 から始まり、上に行くほど
ド派手になり、Tier 15 で全虹（最大演出）になります。

| ファイル | 対象レベル | 目安の累計時間 | 見た目 / エフェクト |
|---------|-----------|--------------|-----------|
| `tier1.png`  | Lv 1–14    | 0〜4時間       | 🥉 ブロンズ |
| `tier2.png`  | Lv 15–24   | 4〜12時間      | 🥉 ブロンズ |
| `tier3.png`  | Lv 25–34   | 12〜23時間     | 🥈 シルバー |
| `tier4.png`  | Lv 35–49   | 23〜48時間     | 🥈 シルバー |
| `tier5.png`  | Lv 50–64   | 48〜82時間     | 🥈 シルバー |
| `tier6.png`  | Lv 65–79   | 82〜125時間    | 🥇 ゴールド |
| `tier7.png`  | Lv 80–94   | 125〜178時間   | 🥇 ゴールド |
| `tier8.png`  | Lv 95–109  | 178〜239時間   | 🥇 ゴールド |
| `tier9.png`  | Lv 110–129 | 239〜335時間   | 💎 クリスタル ✨ 氷の発光 |

- レベルに上限は無いので `tier15.png` は Lv290 以降ずっと使われます（永遠の頂点）。
- 区切り（minLevel）・色・エフェクトは `src/lib/level.ts` の `LEVEL_TIERS` で調整できます。
- 元画像を差し替えたら `python3 scripts/slice_level_icons.py` で再スライスしてください。
